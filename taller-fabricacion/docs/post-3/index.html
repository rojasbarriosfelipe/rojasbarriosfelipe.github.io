2 - Mecanizado CNC

Mecanizado CNC es una manera que se ocupa para cortar materiales con máquinas herramienta controladas numéricamente por computadoras.

El proceso de mecanizado CNC interpreta el modelo CAD 3D y transfiere los datos a la máquina CNC. La principal ventaja de este proceso es que las piezas producidas por CNC pueden reemplazar los prototipos producidos por fabricación aditiva con materiales de calidad de producción. Este método permite imprimir las piezas en los materiales utilizados para la producción.

El mecanizado CNC es ideal para la creación rápida de prototipos de piezas de plástico y metal de alta calidad que requieren el más alto grado de precisión, acabado superficial y propiedades del material.

Las máquinas más frecuentes para CNC

Fresadora

Una fresadora es una máquina para desbastar, cortar o (o incluso) perforar diferentes materiales sólidos (generalmente metales). Esto se logra mediante la rotación de alta velocidad de la herramienta y el movimiento del eje (componente o mecanismo) para eliminar las virutas.

Este tipo de tecnología de rotura de viruta que se utiliza para fabricar piezas se denomina mecanizado. Una fresa es una herramienta que se utiliza en una fresadora. La herramienta contiene uno o más bordes cortantes y el borde cortante desgarra las virutas de metal cuando la herramienta gira. Pasos de Mecanizado CNC

A continuación se mostrará cómo funciona el programa por el cual se dan las instrucciones para que la máquina CNC corte según el resultado que queremos.

Paso 1

Definir área de trabajo: Plancha de Terciado. (selección en SKETCH, con visualización en TOP), se selecciona el plano XY para crear un cuadrado de 1220X1220 mm (media plancha de terciado).

Paso 2

Sobre el cuadrado se realiza un EXTRUDE de 15mm hacia arriba, luego con MOVE/COPY se mueve la silla sobre la plancha por capas, cambiándolas al plano horizontal. Para esto se usa el comando ALIGN. (usar FLIP para que todas las piezas queden en la misma área de trabajo)

Paso 3

Luego se deber ir a DISEÑO>MANUFACTURA, en SETUP se escoge la maquina Autodesk CNC de 3 ejes genérica.

Paso 4

En Stock>from solid se selecciona la plancha anteriormente creada y las piezas a cortar. Para esto se debe apagar la plancha y luego en “orientación” clickear en “Select Z axis/plane & X axis”. Luego se prende la plancha y se orienta el área de trabajo con el eje X frente a nosotros y el eje Y hacia el fondo (y Z como altura).

Paso 5

Para realizar el 2D contour, se seleccionan las aristas de la base y la herramienta de corte; FRESA de 6mm (flat end mil). Aparecerán unas líneas que indican la trayectoria de la FRESA para realizar el mecanizado. *botón derecho>simular

Paso 6

Se debe editar la profundidad de la plancha en Contour2D>passes>multiple dephts y se debe cambiar el maximun roughing a 3mm. Ahora la FRESA recortara la plancha en 5 pasos.

Paso 7

Usando el comando 2D POCKET, se vuelve a seleccionar passes>multiple dephts> y cambiar el máximum roughing a 3mm. Esto rebajará el área interna seleccionada.

Paso 8

Luego, para realizar el área externa, se vuelve a seguir la sucesión de pasos indicada en el PASO 6, que indica el trayecto de la herramienta. Finalmente, al terminar quedara la pieza suelta por lo que para evitar que se rompa la plancha se deben ajustar los TABS en width: 6mm ; Height: 3mm ; Distance: 350mm.

Paso 9

Se hace una simulación final, en la que se mostraran todos los pasos anteriores con la pieza mecanizada.
